## 7. Implementation Requirements

### 7.1. 개발 환경

'협업의 민족'을 개발하기 위해 사용하는 환경이다.

**H/W Platform Requirements**

* RAM : 16GB 이상
* Processor : 12th Gen Intel(R) Core(TM) i5-1240P

**S/W Platform Requirements**

* **OS**
    * Windows 10/11, macOS, 12.0+
* **IDE**
    * IntelliJ IDEA UItimate
    * VS code
* **Programming Language**
    * Java 17 이상 (Spring Boot 기반)
    * Java Script (ES6+) (React 기반)
* **Framework & Library**
    * Spring Boot 3.x+
    * React 19.x+
* **Build & Package Tools**
    * Gradle
    * npm 9.x+ 및 Node.js 22.x+
* **Version Control**
    * git 2.44+
* **Design/Modeling Tools**
    * Figma
    * StarUML

### 7.2. 사용자 환경

최종 사용자가 '협업의 민족' 웹사이트에 접속하기 위한 최소 환경이다.

* **Supported Browsers**
  * Google Chrome
  * Microsoft Edge
  * Apple Safari

# 8. Glossary

| 이름 | 설명 |
| :--- | :--- |
| 협업의 민족 | 사람들의 협업을 도와주기 위한 웹 페이지로, 사람들 간의 시간 조율, 프로젝트별 일정 관리, 공지사항, 게시판 등의 기능을 가지고 있는 본 프로젝트의 이름이다. |
| when2meet | when2meet은 협업할 때 팀원들의 가능한 회의 시간을 찾을 수 있도록 도와주는 간편한 일정 조율 도구 |
| 토큰 | 서버가 각 클라이언트를 누군지 정확히 구별할 수 있도록 유니크한 정보를 담은 암호화 데이터 |
| 참여코드 | 클라이언트가 프로젝트에 참여하기 위해 입력하는 프로젝트의 유니크한 데이터 |
